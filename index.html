Start DJing right now!

Don't waste time (and money) installing DJ software and configuring your system. Simply plug the USB key, restart the computer and start DJing without a hassle!

Powered by Mixxx DJ software.

Features

  • Open-Source

    Every component of TuXXX is Free Software. You can build it on your machine or fork it!

  • Compatibility

    Unlike certain high-end DJ systems, TuXXX doesn't vendor-lock you to any hardware. You can use nearly any sound-card, time-coded vinyl and lot of DJ controllers available on market.

  • Profi Sound

    Although TuXXX is completely free, the real-time kernel and JACK audio layer provide very low-latency audio output.

Contributing

If you think that something should be changed or fixed, you're very welcome to tell us about it! You can use GitHub issue tracking system, or send us a pull-request. If you're not developer, you can also write us an e-mail!
